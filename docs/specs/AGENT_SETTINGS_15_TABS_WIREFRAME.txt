    optional section headers for grouping in the UI.

    control is hidden or disabled with honest copy — otherwise omit tab until
    wired (per plan_iam_dashboard_v2: no mock functionality).

--------------------------------------------------------------------------------

DO NOT:
  - Change vault API routes or payload shapes in this pass.
  - Re-skin provider cards unless tokens only (CSS vars).
  - Move secrets to a different tab without user testing.

    General (same state hooks, same parent context providers).
