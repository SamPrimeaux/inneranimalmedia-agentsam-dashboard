# Brand guidelines

Inner Animal Media — visual and naming consistency for web and dashboard.

## Identity

- Use approved wordmark and color tokens; prefer CSS variables / theme system over hardcoded hex in UI code.

## Themes

- Dashboard themes (e.g. Meaux Storm Gray, Meaux Glass Blue) are user-selectable; do not break theme variables when adding components.

## Updates

- Major brand changes should be reflected here and in design tokens before wide rollout.
