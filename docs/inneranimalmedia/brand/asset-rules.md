# Asset rules

## Images and media

- Optimize for web; prefer modern formats where pipeline supports them.
- Naming: consistent, lowercase-with-hyphens or project convention; avoid spaces in deployed keys.

## Fonts

- Dashboard loads defined font stacks from shell/CSS; do not add ad-hoc third-party font loads without review.

## Legal

- Only use assets we have rights to ship; document licensed packs in product notes when relevant.
