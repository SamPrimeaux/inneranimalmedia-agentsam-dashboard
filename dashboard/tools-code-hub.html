Agent code pipeline

When Agent Sam (or any agent) writes or builds code, the first landing zone is the TOOLS bucket under code/. This page documents the promotion path until your dashboard replaces it.

1 Write to TOOLS

Artifact lands at tools / code / <file> via r2_write (binding TOOLS).

2 Public URL

Share https://tools.inneranimalmedia.com/code/… for review (custom domain on bucket).

3 Approval

Human approves content and promotion intent (no auto-prod without gate).

4 Sandbox R2

Upload or sync into agent-sam-sandbox-cidi for live validation on the sandbox worker.

5 Production

After sandbox check, promote to agent-sam and production URLs per CIDI rules.
